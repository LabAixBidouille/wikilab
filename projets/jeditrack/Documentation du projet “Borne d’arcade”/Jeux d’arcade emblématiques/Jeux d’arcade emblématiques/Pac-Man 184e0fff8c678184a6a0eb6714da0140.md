# Pac-Man

Concepteur: https://fr.wikipedia.org/wiki/T%C5%8Dru_Iwatani
Description: L'un des jeux d'arcade les plus emblématiques de tous les temps, Pac-Man a été un véritable phénomène culturel dans les années 1980. Le but du jeu est simple : manger tous les pac-gommes dans le labyrinthe tout en échappant aux fantômes.
Développeur: Namco
Sortie arcade: 22 mai 1980
