Pour préparer votre utilisation de la Trotec Laser Speedy 400 depuis chez vous, nous vous recommandons l'utilisation du logiciel Inkscape, que vous pourrez télécharger sur le [site du logiciel](https://inkscape.org/fr/). De nombreux tutoriels existent sur le web. Veillez à respecter le code couleur indiqué dans le tuto renseigné ci-dessus pour ne pas avoir de mauvaise surprise.

Ce logiciel est celui recommandé mais vous pouvez utiliser d'autres logiciels de dessin vectoriel si leur utilisation vous est plus pratique.
