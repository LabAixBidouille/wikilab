# Sauver les bébés dinosaures

Objectifs: Créer un jeu de style collector mettant en vedette un dinosaure déterminé à sauver les bébés dinosaures
Aspects créatifs et ludiques abordés: Animer des personnages, gérer la vitesse, créer un fond animé, collecter des points
